# ClaudeSample

ClaudeSampleプラグイン

## 動作環境

- EC-CUBE 4.3系

## インストール方法

### オーナーズストアからインストール

1. EC-CUBE管理画面にログイン
2. オーナーズストア > プラグインを選択
3. ClaudeSampleを検索してインストール

### コマンドラインからインストール

```bash
bin/console eccube:plugin:install --code=ClaudeSample
bin/console eccube:plugin:enable --code=ClaudeSample
```

## アンインストール方法

```bash
bin/console eccube:plugin:disable --code=ClaudeSample
bin/console eccube:plugin:uninstall --code=ClaudeSample
```
